## User Direction

At session start, read the following files for context on the user's goals, challenges, and beliefs:
- `~/.pi/agent/direction/goals.md` — what the user is working toward
- `~/.pi/agent/direction/challenges.md` — current constraints and blockers
- `~/.pi/agent/direction/beliefs.md` — core values shaping decisions

Use this to prioritize work, filter recommendations, and flag misalignment with stated goals.
